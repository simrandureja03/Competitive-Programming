# C. Factors of Factorial

### Description
You are given an integer **N**.  
Find the number of positive divisors of **N!**, modulo **10⁹ + 7**.

---

### Input
The input is given from standard input in the following format:

- A single integer **N**  
  `(1 ≤ N ≤ 1000)`

---

### Output
Print the number of positive divisors of **N!**, modulo **10⁹ + 7**.

---

### Sample

**Input:**
3


**Output:**


4


**Explanation:**  
3! = 6  
The positive divisors of 6 are: 1, 2, 3, and 6.  
Hence, the answer is 4.

---
