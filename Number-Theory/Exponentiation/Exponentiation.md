# Exponentiation

### Description
You are given several calculations. For each calculation, you are given two integers **a** and **b**, and you need to compute **aᵇ modulo 10⁹ + 7**.

Note: For this problem, assume that **0⁰ = 1**.

---

### Input
- The first line contains an integer **n** — the number of calculations.
- Then follow **n** lines, each containing two integers **a** and **b**.

---

### Output
For each calculation, print the value of **aᵇ modulo 10⁹ + 7** on a new line.

---

### Sample

**Input:**
3
2 3
5 0
10 10

makefile
Copy code

**Output:**
8
1
999999937
