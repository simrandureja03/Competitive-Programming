# D. Buying Shovels

### Description
You want to buy exactly **k** shovels.  
A shop sells shovels in packages of size **x** (1 ≤ x ≤ k).  
Each shovel package of size **x** costs **x** coins.

Find the minimum number of coins you need to pay to buy exactly **k** shovels.

---

### Input
The first line contains a single integer **t** — the number of test cases.  
Each of the next **t** lines contains a single integer **k**.

**Constraints:**
- 1 ≤ t ≤ 1000
- 1 ≤ k ≤ 10⁹

---

### Output
For each test case, print one integer — the minimum number of coins needed.

---

### Sample

**Input:**
5
1
2
3
4
5


**Output:**
1
2
3
2
5
